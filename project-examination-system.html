Examination Center Management System

System Design Desktop app

Project Overview

An academic project focused on developing a comprehensive desktop management application for an examination center. The system handles key operations such as candidate registration, exam scheduling, and the management of results and certificates. The application was built using Windows Forms (C#) with a Microsoft SQL Server database.

My Role & Responsibilities

Role: Business Analyst, WinForm Developer

  • Process Analysis: Analyzed the examination center's business processes, including registration, payment, exam extensions, and certificate issuance.
  • UML Modeling: Modeled the system's functional requirements and workflows by creating UML diagrams, specifically Use Case Diagrams and Activity Diagrams.
  • UI & Desktop Development: Designed the User Interface (UI) and developed a complete desktop application (Windows Forms) to manage the center's operations.
