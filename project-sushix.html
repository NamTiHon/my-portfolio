SushiX - Ordering Database & Platform

Database Design Web PoC

Project Overview

An academic project focused on analyzing business requirements and designing a complete database solution for a restaurant's online ordering system. The project's core focus was on translating business requirements into a well-structured and reliable database.

My Role & Responsibilities

Role: Business Analyst, Database Designer, Web Developer

  • Business Analysis: Analyzed the restaurant's operational flows (menu management, customer ordering, billing, inventory, and branch distribution).
  • Database Engineering: Translated business requirements into a technical database schema. Designed the Entity-Relationship Diagram (ERD) and performed 3NF Normalization to ensure zero data redundancy.
  • Proof-of-Concept Web Development: Developed a functional web prototype using ReactJS for the front-end interface, backed by a SQL Server database to demonstrate transaction consistency.
