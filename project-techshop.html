TechShop - Full-Stack E-Commerce

Scrum Team Full-stack

Project Overview

A full-stack e-commerce platform designed for selling technology products, developed as part of an Introduction to Software Engineering course. This project was managed using Scrum methodologies, with a technical stack featuring ReactJS for a responsive user interface and MongoDB as the back-end database.

My Role & Responsibilities

Role: Scrum Business Analyst, Web Developer

  • Agile Product Ownership: Collaborated within a Scrum team to break down broad business requirements into actionable tasks and user stories. Managed the product backlog using Jira.
  • Responsive UI/UX Coding: Developed the shopping interface using ReactJS, ensuring full responsiveness, smooth transitions, and a pleasant user ordering experience.
  • Requirements Alignment: Acted as the bridge between development task prioritization and UX mockup design wireframes, aligning Figma components with backend database constraints.
